1.	使用 Python 語言，並引入必要的模組：NumPy、Streamlit、Matplotlib 和 Scikit-Learn。
2.	設置 Streamlit 頁面標題與描述，說明這個應用展示的是非圓形分布的數據集，並包含可調整的距離閾值滑桿。
3.	使用 NumPy 生成 600 個隨機數據點 (x1 和 x2)，其中 x1 和 x2 分別具有不同的方差 (variance_x1=20，variance_x2=5)，以生成橢圓形分布。
4.	添加 Streamlit 的滑桿 (st.slider) 來調整距離閾值，範圍為 1.0 至 10.0，初始值為 4.0。滑桿值會影響數據標籤的分配。
5.	計算每個數據點到原點的距離，並根據滑桿閾值分配標籤 Y：當距離小於閾值時標籤為 0，否則為 1。
6.	定義高斯函數，將 x1 和 x2 映射到第三維 x3，使數據具有非線性特徵。
7.	將 x1、x2 和 x3 作為特徵向量，使用 LinearSVC 訓練支持向量機模型來找到分離超平面。
8.	繪製兩個圖：
•	3D 散點圖：使用 Matplotlib 建立 3D 圖，顯示分類標籤的分布和分離超平面。數據點的顏色區分標籤 0 和 1。
•	2D 投影圖：展示數據集在 feature plane 上的 2D 投影，使用顏色標記兩個標籤。
9.	使用 st.pyplot(fig) 將 3D 和 2D 圖表嵌入到 Streamlit 應用中。隨著滑桿調整距離閾值，圖表會即時更新，以反映新的分類結果和超平面分布。
